 BMI Calculator App

A simple Flutter application that calculates Body Mass Index (BMI) based on user selections.

Installation & Run

Follow these steps to run the project on your device:

1. Clone the repository
```bash
git clone https://github.com/Yalda-Ashrafi/bmi-calculator-app.git
2. Go to the project folder
cd bmi-calculator-app
3. Install dependencies
flutter pub get
4. Run the app
flutter run

Features
Enter height and weight
Calculate BMI instantly
Get health category (Underweight, Normal, Overweight)
Clean and modern UI
